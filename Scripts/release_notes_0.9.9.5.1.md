# MiMiNavigator v0.9.9.5.1

Breadcrumb hover lens clarity and release metadata update.

## Changed

- Breadcrumb hover segments now use a stronger convex 3D lens with layered highlight, inner shadow, and raised drawing order above adjacent breadcrumb items.
- Release metadata updated to version `0.9.9.5.1`, build `122`.

## Fixed

- Hover enlargement now uses a real font-size change instead of scaling the rendered button, keeping breadcrumb text sharper.
