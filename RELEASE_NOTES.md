# MiMiNavigator v0.9.9.5.4

Cloud Share+Link privacy and collision-resistance update.

## Highlights

- Google Drive and Dropbox links now use `https://spoo.me/mimiNavi_<14 random Base62 characters>`.
- The shorter branded prefix remains recognizable without exposing a predictable sequence.
- The random suffix provides roughly 83 bits of alias space and makes collisions impractical.

## Changed

- Replaced the previous two-character UUID prefix with 14 characters generated by `SystemRandomNumberGenerator`.
- Restricted aliases to URL-safe Base62 characters instead of punctuation that can require escaping.
- Kept collision retries in place so a rejected alias is replaced automatically.

## Documentation

- Added Cloud Share+Link developer design notes covering provider flows, OAuth storage, shortener behavior, and testing.
- Updated contributor and AI-agent guidance with the permanent alias format and security constraints.
- Updated release metadata to version `0.9.9.5.4`, build `123`.

## Validation

- Added a regression test that validates 1,000 generated aliases for format, URL safety, and uniqueness.
- Unit-test target passes on macOS.

## Download

The DMG is signed, notarized by Apple, and includes an Applications shortcut for drag-to-install.
